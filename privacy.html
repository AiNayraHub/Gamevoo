Privacy Policy

1. Information We Collect

Hum aapka basic data collect karte hain jaise Email, Username aur Device ID taaki hum aapke rewards aur coins ko safe rakh sakein. Hum aapka koi bhi personal password ya sensitive data save nahi karte.

2. How We Use Data

Aapka data sirf account management, withdrawal processing aur personalized offers (Offerwalls) dikhane ke liye use kiya jata hai.

3. Third-Party Services

Hum third-party networks (CPAGrip, Monlix, etc.) ke saath kaam karte hain. In networks ki apni alag privacy policies ho sakti hain jab aap unke offers complete karte hain.

4. Data Security

Hum Supabase encryption ka use karte hain taaki aapka balance aur profile details hackers se safe rahein.
